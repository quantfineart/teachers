# Teacher's Workspace

A teaching assistant that remembers how you work and gets better over time. Lesson plans, grading, parent comms, and more — in any language, for any curriculum.

No programming required. If you can type a message, you can use this.

## Why This Instead of ChatGPT?

ChatGPT forgets everything between conversations. This tool remembers — your name, your curriculum, your grading system, your class size, your preferred format. It creates real files you can print, project, and share. The more you use it, the less you explain.

## What You Need

- A **laptop or desktop** (not a phone)
- **Internet** while using it (but files it creates work offline)
- A **Claude account** — paid plan from [Anthropic](https://www.anthropic.com), roughly $20/month as of 2025
- About **30 minutes** for first-time setup — ask a tech-savvy colleague to help if you've never used a terminal before

## Setup

1. **Download** — click the green "Code" button above → "Download ZIP" → extract the folder
2. **Install Claude Code** — follow the guide at [Claude Code](https://docs.anthropic.com/en/docs/claude-code/overview). You'll paste one command into your terminal (Mac: search "Terminal"; Windows: search "PowerShell")
3. **Start** — in your terminal, go to the folder and type `claude`:
   ```
   cd Desktop/teachers
   claude
   ```
4. **Say hello** — type "Help me set up my workspace" and you're off

> You'll see technical-looking text sometimes — that's Claude working behind the scenes. Just focus on the messages addressed to you.

## What It Does

- **Lesson plans** — single lessons, weekly plans, or full term/year plans
- **Grading** — report card comments, rubrics, comment banks, batch grading for your whole class
- **Parent communication** — emails, newsletters, meeting notes
- **Everything else** — worksheets, quizzes, revision guides, visual schedules, and more

Type `/lesson` for plans, `/grade` for grading, or just describe what you need. Claude will suggest **new shortcuts** as it learns your patterns — your workspace grows with you.

## Your Privacy

Claude never saves student names to your profile. We recommend using pseudonyms ("Student A") instead of real names.

**For administrators:** Messages are sent to Anthropic's servers to generate responses. Review [Anthropic's privacy policy](https://www.anthropic.com/privacy) and check your school's data protection policies (GDPR, FERPA, etc.) before use. Files Claude creates stay on the teacher's computer only.

## Tips

- **Drop files in your folder** and Claude will use them — templates, syllabi, past papers, class lists. PDF and text files work best (save Word docs as PDF first).
- **Share outputs** by email, USB, Google Classroom, or any platform your school uses.
- **Give a co-teacher their own copy** of this folder — each teacher needs their own.
- **Update the template** by replacing "CLAUDE.md" with the new version. Never replace "teacher-profile.md" — that's your data.
